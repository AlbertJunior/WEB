Abous us

We devised a Web application for visualizing and creating statistics based on terrorist attacks in the last 50 years. It features a registration system for users, statistics generated based on custom options and a map visualizer. The app uses and processes hundred of thousands of terrorist attack records.

Having the records of terrorist attacks requested, the application is able to generate all kinds of statistics, from piecharts showing the most preferate countries as targets for attacks, to histograms and representations that can depict the success rate of the attack, the amplitude of damages made and so on. The user can get custom statistics based on a completion of a form. Based on the given input, by the user, the application is able to make specific queries to the database.

These powerful illustrations can be really handy for understanding any patterns involving terrorist attacks, and can be featured on thematic presentations.

Another feature is the world map, modified such that it illustrates the kind of attacks requested by the user. The exact locations of the attacks will be pin-pointed based on this geographic coordinates.

Any graphic element generated (map or statistics) can be externally exported to the formats: png, jpg. The attacks with all the information included can be exported in csv format.
